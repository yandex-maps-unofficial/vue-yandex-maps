# Миграция с версии 2.x

- Минимальная версия Node поднята до 20
- Удалена поддержка Vue 2, Nuxt 2 и Nuxt Bridge
- Минимальная версия Nuxt поднята до 3.12
- Добавлена поддержка указания `runtimeConfig.public.yandexMaps` в Nuxt
- Удалён метод `useYMapsLocationFromBounds`. Вместо него, используйте `getLocationFromBounds` (изменений нет)
- События `input` удалены, ровно как и `value`, остались только `modelValue` и `update:modelValue`
- Выполнен переход на script setup
  - У многих компонентов теперь будут подтягиваться top-level атрибуты, навешанные прямо на компонент (data-v-*, классы, и т.д.)
  - Многим компонентам добавлены классы
  - Убраны костыли, убраны слоты там, где их быть не может
- Удален компонент YandexMapDefaultSatelliteLayer
- Добавлена поддержка `index` в `YandexMapMiniMap`, 
- Добавлен тип `YandexMapPopupMarkerSettings`

```typescript
new ymaps3.YMapDefaultSatelliteLayer({visible: true})
```
